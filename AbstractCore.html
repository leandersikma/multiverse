Abstract Class: AbstractCore

The Abstract Class 'AbstractCore' is an upper class of the API and Core class. Those class extends the AbstractCore class mainly to adapt the Route() function. This function is used to parse the URL, by using the Handling class, and call the specific class and function.

AbstractCore constructor

The constructor in AbstractCore class creates new instances of the following classes: Handling, Loader and OutputHandler. Each class is respectively approachable by using $this->handling, $this->load and $this->output. More about those classes is explained in later chapters.

Routing

The Route() function inside the AbstractCore class is used to send the URL request further by getting the controller and method name from the URL. The Route() function checks if the controller exists, if yes it creates a new instance of the controller. After the instantiation, the function checks if the method is set. If yes, the function use that method defined by the URL. If no, Route() does a fallback on the function index() inside the called controller.

After all, the function call_user_func_array() is used to call the controller and method with parameters as follow: 

$this->output->appendOutput = call_user_func_array(array(this->controller, $this->action_name), array($this->handling->parameters));

The output is added to the Singleton output class.
