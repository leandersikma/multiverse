Multiverse

And API and web framework packed in one thing!

Welcome to Multiverse!

Multiverse offers you a structured way to simply develop API centric applications. With the API first principle, you are always ready to support new devices such as Android apps, iOS apps and third-party apps. Multiverse comes with a neat debug tool to debug your newly created API fast and precisely.

Multiverse is ideal for:

  • Building back-end systems for mobile apps.
  • Building web apps to support directly developers to build third-party apps on top of your web app.

The Multiverse open source project is growing rapidly. We improve the framework often and we challenge you to contribute with the project to build a strong API centric development framework! It is the future of web and app systems and you can contribute to this!

-- Multiverse team
