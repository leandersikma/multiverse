Application Controllers

The application controllers are the part of Multiverse where you can build your back-end system and front-end web app. You could place you controllers inside the application/controllers/ folder.

Anatomy of a controller

A custom Multiverse controller should always extends the MV_Controller and override the parent constructor function. Example of a custom controller:

class Contacts extends MV_Controller {
	
	public function __construct() {
		parent::__construct();
	}
	
	public function index() {
		$this->read();
	}
	
	public function read() { }
	
	public function create() { }
	
	public function update() { }
	
	public function delete() { } 		
}
