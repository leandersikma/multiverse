Class: Database

The 'Database' class is used inside the API to connect with the database and to perform SQL queries. This Database class is a wrappe to custom PDO queries. In the near future, this class is going to transform to an Active Record like Database class. For now, it holds some simple functions for database queries.

Connection Database

The connection with your MySQL database is established inside the constructor function. Multiverse is using PDO to perform a safe connection with your database. If you want to configure the settings for your connection, see: API Models.

Select function: selectFields()

The select function is a simple select function which gives you an array with results back. You could use this function by defining a table (to select) and optionally a where/orderBy functionality.

$result = $this->db->selectFields($table, $_where=null, $_orderBy = null);
print_r($result); // Array of selected results from $table;

Update function: update()

You can use the update function to update records in your database. Use the update function as follow:

$updated = $this->db->update($table, $newData, $where);
$table
Which table needs to be updated.
$newData
An array with newData. Example: array("username"=>"Johan", "password" => "md5hash");
$newData
An array with a where value. Example: array("username", "Johan");

Insert function: insert()

By using this function you could insert data into the database. The format for using update is:

$insertedData = $this->db->insert($table, $data);
$table
Which table need the data inserted.
$newData
An array with new data. Example: array("username"=>"Johan", "password" => "md5hash");
